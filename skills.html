¿Qué es una Skill en AgentHub?

Una skill es una capacidad funcional modular que se puede asignar a un agente para ampliar lo que puede hacer. Cada skill encapsula una integración o comportamiento específico —como sincronizar con un CRM, redactar correos o analizar sentimiento— y puede ser contratada de forma independiente por los clientes. Los agentes adquieren sus habilidades combinando múltiples skills según las necesidades del negocio.
